1. Вызов через имя класса: Так как метод printVars() является статическим, его можно вызывать напрямую через имя класса без создания экземпляра.
пример
A.printVars();

2.Вызов через объект класса: Хотя это не обязательно, статический метод может быть вызван через объект класса, но это не рекомендуемый способ, так как метод все равно будет привязан к классу, а не к объекту.
пример
A obj = new A();
obj.printVars();

3. Вызов из другого статического метода в том же классе: Если мы находимся в статическом методе того же класса, можем вызвать printVars() напрямую без указания имени класса.
public static void anotherStaticMethod() {
    printVars();
}
